# Language & Communication Chat Tests

This file now points to the shared template-driven prompt pack.

## Use These Prompt Packs

- Main guide: `tests/README.md`
- English templates: `tests/en/library-chat-prompts.md`
- German templates: `tests/de/library-chat-prompts.md`

## Quick Usage

1. Choose a target markdown file in `library/`.
2. Copy one template from `tests/en/library-chat-prompts.md`.
3. Replace placeholders (`<LANG>`, `<CHAPTER_SLUG>`, `<FILE_SLUG>.md`, `<TASK_INSTRUCTION>`).
4. Paste into chat while the library context is active.
5. Validate output against the acceptance checklist in that prompt.
